配置页面布局 — 你倾向哪种？

四个 Provider（LLM / TTS / 文生图 / 图生视频）统一管理

A

Tab 切换式

顶部四个 Tab，每个 Provider 一个面板。点击切换，表单字段随 Provider 类型变化。适合 Provider 参数差异大的场景。

持久化：一键保存全部四个 Tab

B

单页纵向排列

四个 Provider 区块纵向堆叠，一滚动看完。每个区块独立显示当前选中 Provider 和关键参数。

持久化：每个区块独立保存按钮

C

卡片网格

2×2 卡片网格，每张卡片代表一个 Provider 类型。点击卡片弹出侧边栏或 Modal 进行编辑。

持久化：Modal 内保存，卡片实时反映当前选中 Provider
