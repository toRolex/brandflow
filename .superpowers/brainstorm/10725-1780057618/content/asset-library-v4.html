智能素材库 — 完整设计方案

所有交互状态、组件结构、数据流一览

一、组件树

SmartAssetLibrary (Tab 页)
├── AssetStatsBar          统计卡片行（总切片/可用/禁用/源视频数）
├── AssetToolbar           上传按钮 + 分类筛选 + 关键词搜索
│   ├── UploadZone         拖拽/点击上传区（展开/折叠）
│   └── PendingList        待入库视频列表（可移除单项）
├── AssetGrid              素材网格
│   ├── AssetCard[]         单张卡片（缩略图/分类/置信度/时长/使用次数）
│   │   └── 点击 → AssetPreviewPanel（弹出侧边面板）
│   └── 多选模式（Ctrl/Shift）
├── BatchActionBar          批量操作栏（选中后浮现）
└── IndexProgress           索引进度条（处理中显示）

二、状态机

上传区
折叠 → 展开 → 有文件
有文件时显示「确认入库」按钮
索引
空闲 → 处理中 → 完成
完成 3 秒后自动消失
选择
无选中 → 有选中
浮出批量操作栏

五、数据流

┌─────────────────────────────────────────────────────┐
│                    React State                       │
│  assets: AssetRecord[]    indexed clips from DB     │
│  pendingFiles: File[]     files not yet submitted   │
│  indexStatus: 'idle' | 'processing' | 'done'        │
│  selectedIds: Set multi-select              │
│  categoryFilter: string                             │
│  keywordSearch: string                              │
│  uploadExpanded: boolean                            │
│  previewAsset: AssetRecord | null                   │
└─────────────────────────────────────────────────────┘
                        │
         ┌──────────────┼──────────────┐
         ▼              ▼               ▼
  GET /assets/indexed  POST /assets/upload  POST /assets/index
  获取所有已索引切片     上传新视频文件       对 pending 文件触发索引
                        返回文件名列表         返回 {indexed, skipped, clips}

六、后端新增 API（3 个端点）

方法 路径 说明 返回
GET /api/projects/{id}/assets/indexed 查询已索引素材 AssetRecord[] + stats
POST /api/projects/{id}/assets/index 触发增量索引 {indexed:2, skipped:1, total_clips:8}
PATCH /api/projects/{id}/assets/{asset_id} 更新状态/批量操作 {updated: 3}
