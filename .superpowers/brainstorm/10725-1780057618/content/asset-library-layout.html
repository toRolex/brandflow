核心交互流程

上传 → 索引 → 分类

  1. 用户拖拽/点击上传源视频
  2. 点击「一键索引」触发后端 AssetIndexer
  3. FFmpeg 场景检测切分 + VisionClient AI 分类
  4. 切片入库 SQLite，按分类展示

筛选 → 检索 → 复用

  1. 按分类下拉筛选或关键词搜索
  2. 查看每个切片的置信度、使用次数
  3. 手动禁用低质量切片
  4. Job 生产时自动调用 AssetRetriever 匹配
