# QA 验收文档

本目录存放 Brandflow 部署与功能验收的 QA 清单和报告模板。

## 文件说明

| 文件 | 用途 |
|------|------|
| `deploy-checklist-0.7.x.md` | 0.7.x 版本 Windows 部署包验收清单，覆盖健康检查、前端、产品配置、素材上传、工具缺失提示、日志、重启/重复部署。 |

## 如何使用验收清单

1. 按照 `packaging/windows/README.md` 完成部署包解压、工具放置、`init.bat` 初始化、`.env` 配置和前端依赖安装。
2. 运行 `packaging\windows\start.bat` 启动后端和前端。
3. 打开 `deploy-checklist-0.7.x.md`，逐项勾选并记录实际结果。
4. 对“未通过”项，按清单中的“失败时的下一步”排查，必要时查看 `logs/backend.log`、`logs/frontend.log` 和浏览器控制台（F12）。

- 验收人：xxx

| 章节 | 结果 | 备注 |
